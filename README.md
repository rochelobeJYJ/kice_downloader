# KICE Downloader (KICE 기출문제 다운로더) 📥

한국교육과정평가원(KICE) 웹사이트에 등록된 대학수학능력시험 및 6월/9월 모의평가 기출문제(문제지 및 정답지)를 빠르고 간편하게 일괄 다운로드할 수 있는 자동화 데스크톱 애플리케이션입니다.

## 📌 주요 기능

*   **다중 연도 및 시험 일괄 다운로드:** 시작 연도와 끝 연도를 지정하고, 수능/6월/9월 모의평가 중 원하는 시험을 한 번에 선택하여 다운로드할 수 있습니다.
*   **세부 과목별 맞춤 추출:** 탐구 영역처럼 여러 과목이 하나의 ZIP 파일로 묶여 있는 경우에도, 사용자가 체크한 특정 과목(예: `물리학I`, `생명과학II`, `생활과 윤리`)의 PDF 파일만 쏙쏙 뽑아 추출합니다.
*   **과거 교육과정 완벽 지원:** '수리', '언어', '생물', '국사' 등 이름이 바뀌었거나 옛날 인코딩(CP949) 방식으로 저장된 과거 기출문제들도 자동으로 인식하여 정상적으로 다운로드합니다.
*   **[NEW] 꼼꼼한 평가원명칭 오기 방어 로직 탑재:** 파일명 규칙이 일관되지 않은 연도(예: '지구과학1', '지구과학 I' 혼용 등)에서도 누락 없이 정확하게 정답지를 캐치하여 다운로드합니다.
*   **깔끔하고 직관적인 GUI:** 사용자 친화적인 그래픽 화면을 제공하며, 사용자 편의를 위해 가변적인 수요에 맞게 초기 시작 시 모든 과목 선택이 프리셋(해제)된 상태로 시작됩니다.
*   **스마트한 파일명 자동 변환:** 복잡하고 불규칙한 원본 파일명을 설정한 템플릿(예: `2024_수능_국어_문제지.pdf`)에 맞춰 일관성 있게 자동 변환하여 저장합니다.
*   **안전 다운로드 모드 지원:** 대량의 파일을 안전하게 받기 위해 파일 간 **랜덤 딜레이(0.5초~1.5초)**를 부여하는 등 안전 장치가 기본 탑재되어 있습니다.

## 🚀 설치 및 실행 방법

### 1. 요구 사항 확인
- **Python 3.8 이상**이 설치되어 있어야 합니다.
- 필요한 외부 라이브러리 목록은 `requirements.txt`에 기록되어 있습니다. (`requests`, `beautifulsoup4`)

### 2. 라이브러리 설치
명령 프롬프트(CMD) 터미널을 열고 다음 명령어를 입력하여 필수 라이브러리를 설치합니다.
```bash
pip install -r requirements.txt
```

### 3. 프로그램 실행
*   **기본 실행 (GUI 창 띄우기):**
    ```bash
    python main.py
    ```
*   **백그라운드 실행 (콘솔 창 없이 실행):**
    동봉된 `kice_down.pyw` 파일을 더블클릭하면 시커먼 콘솔 창 없이 깔끔하게 앱 화면만 실행됩니다.
*   **실행 파일 빌드:**
    `build_dist.bat` 파일을 실행하면 코드를 손쉽게 단일 실행 파일(`.exe`) 형태로 패키징할 수 있습니다.

## 💡 사용 방법 안내

1.  **연도 범위 설정:** 다운받고자 하는 기출문제의 시작 연도와 끝 연도를 설정합니다. (학년도 기준)
2.  **시험 종류 선택:** `대학수학능력시험`, `6월 모의평가`, `9월 모의평가` 중 원하는 시험을 체크합니다.
3.  **과목 선택:** `공통`, `사회탐구`, `과학탐구` 등의 탭을 이동해가며 다운받고 싶은 과목을 직관적으로 확인하고 체크합니다. (상단의 '전체 선택/해제' 활용)
4.  **저장 경로 지정:** `폴더 선택` 버튼을 눌러 파일이 저장될 위치를 설정합니다. (기본값: 사용자 바탕화면의 `KICE_기출문제` 폴더)
5.  **다운로드 시작:** 우측 하단의 `▶ 다운로드 시작` 버튼을 누르면 작업이 시작됩니다. 하단 상태바 및 실시간 로그를 통해 진행 상황을 모니터링할 수 있습니다.

## ⚙️ 알아두면 좋은 점 (문제 해결)

*   **듣기 대본/음원 제외:** 이 프로그램은 영어 영역의 **문제지와 정답지만**을 추출하며, 용량과 처리 시간을 최적화하기 위해 음원(MP3 등)과 듣기 대본 파일은 다운로드 목록에서 자동으로 제외됩니다.
*   **정답표 통짜 파일:** 과거 연도 중 평가원이 과학탐구나 사회탐구 정답지를 하나의 거대한 통짜 PDF로 올린 경우, 세부 과목으로 쪼개지지 않고 포괄적 대표 영역 이름 하나로 다운로드됩니다. (예: `2011_수능_과학탐구_정답.pdf`)
